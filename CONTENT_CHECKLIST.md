- [ ] **Individual Headshots** (all featured staff)

  - Teachers engaging with students

### Photo Requirements
- **Format:** JPEG or PNG
- **Resolution:** Minimum 1920px on longest side
- **Quality:** High resolution, well-lit, in focus
- **Permissions:** Written consent for all children's faces
- **Diversity:** Represent diverse children and families
- **Authenticity:** Real photos preferred over stock images

### Alternative: Stock Photos
If authentic photos aren't available immediately:
- [ ] Budget for stock photos ($200-500)
- [ ] Curated stock photo collection from:
  - Unsplash (free)
  - Pexels (free)
  - iStock or Shutterstock (paid, higher quality)

## üéâ Content Submission

### How to Submit Content

**Option 1: Google Doc**
- Create a shared Google Doc
- Organize by section (Homepage, About, Programs, etc.)
- Share edit access with development team

**Option 2: Markdown Files**
- Create `.md` files in `/content` directory
- Use clear naming (e.g., `homepage-content.md`, `program-infant.md`)
- Commit to Git repository

**Option 3: Spreadsheet**
- Staff bios, testimonials, FAQs work well in spreadsheet format
- Easier to review and approve in bulk

### Photo Submission
- Use cloud storage (Google Drive, Dropbox)
- Organize in folders by category
- Name files descriptively (`outdoor-playground-1.jpg`, not `IMG_1234.jpg`)
- Include separate document with captions and photo credits
